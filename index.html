spring-simple

This simple example demonstrates how Spring container is configured, instantiated and being used by the application, e.g. example.run.SimpleSpringExample.

  • Application configures the Spring container first, and then it asks the container for a service instance. Finally, the application calls the service instance it got from the Spring container.
  • Spring container is represented by Spring's org.springframework.beans.factory.BeanFactory.
  • Spring container is configured by src/java/resources/applicationContext.xml Spring configuration file.
  • Spring container instantiates example.services.TimeServiceImpl of examples.services.TimeService type (interface).
